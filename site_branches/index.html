O que é Git?

Git é um sistema de controle de versão distribuído, gratuito e de código aberto, utilizado para rastrear alterações em arquivos e gerenciar o histórico de código-fonte em projetos de desenvolvimento de software.

O que é GitHub?

O GitHub é uma plataforma em nuvem que hospeda projetos Git, facilitando a colaboração, compartilhamento e versionamento de código de forma online.
